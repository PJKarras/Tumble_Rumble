Our project contains multiple resource files for our graphics and is best retrieved from GitHub.
In order to run out program you will need the following libraries installed

numpy
pygame version 2.0.0.dev6 (installed with pip install pygame==2.0.0.dev6)

To run the program simply run test.py. The current build was presented with Python version 3.8.0.
